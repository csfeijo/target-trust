No HTML5 foram criadas NOVAS TAGS para evitar o uso excessivo das DIV´s.
Apesar de ainda ser muito comum e necessários usarmos as DIVS, existem outras TAGS que fazem o mesmo papel porém tem um 
signoficado semantico diferente:

MAIN : conteudo principal (não é obrigatória e poucamente usada)
HEADER: cabeçalho
FOOTER: rodapé
NAV: menus / links
ASIDE: conteudo localizado na lateral do site
SECTION: secao de uma pagina
