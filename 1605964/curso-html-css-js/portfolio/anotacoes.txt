==== ELEMENTOS SEMANTICOS NO HTML 5 ====
Que se comportam como DIV´s, ou seja, em bloco.
Mas que possuem nomes condizentes com o seu papel/contexto:

header  : para cabeçalhos
nav     : para menus
aside   : para colunas
section : para secoes
article : para artigos
footer  : para rodapes
